The Pulsar API server is designed with the goal of using object-oriented design to achieve data synchronization across clients. Key features include integration with Angular, extensibility, and selective data syncing that allows clients to maintain object references. These features allowed network integration with my existing canvas game engine (Mallet), although unfortunately implementing canvas interaction wasn't implemented in time for this assignment.
